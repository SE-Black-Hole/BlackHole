Steps to run use cases:
- open up the command line / Powershell
- Python must be installed
- install python packages by typing in "pip install -r requirements.txt" in command line
- to run Use Case 1, type in "python mixue_bao_use_case_1.py" in command line
- to run Use Case 2, type in "python mixue_bao_use_case_2.py" in command line
- will need to wait a few seconds to connect to database in each use case
